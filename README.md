# Books Scraper

Парсер для сбора данных о книгах с сайта https://books.toscrape.com

## Цель проекта
Сбор информации о книгах и ее сохранение в файл:
- название
- цена
- рейтинг
- количество в наличии
- описание
- дополнительные характеристики из таблицы Product Information

## Инструкции по запуску

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```
### 2. Запуск парсера
```bash
python scraper.py
```
### 3. Запуск тестов
```bash
pytest tests/ -v
```

## Список используемых библиотек
- `requests` - отправка HTTP-запросов к сайту
- `beautifulsoup4` - парсинг HTML-страниц
- `schedule` - планирование автоматического запуска парсера
- `pytest` - фреймворк для тестирования
